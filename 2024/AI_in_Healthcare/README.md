Αναφέρονται επιγραμματικά οι βασικές τεχνικές Μηχανικής Μάθησης, οι παραλλαγές των νευρωνικών δικτύων που αφορούν τη Βαθιά Μάθηση, καθώς και πώς θα μπορούσε η Τεχνητή Νοημοσύνη να προσφέρει στον τομέα της υγείας. 
Στην συνέχεια δίνουμε ένα παράδειγμα και αναλύουμε τα βήματα ώστε να εκπαιδευτεί ένα υποθετικό μοντέλο.
